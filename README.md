# 🚀 Personal Portfolio Website

🌐 **Live Demo:** [Kamalraj's Portfolio](https://kamalraj-s.github.io/Kamalraj-s-portfolio/)

## 📌 About the Project
This is a polished **personal portfolio website** built with **vanilla HTML, CSS, and JavaScript**. It showcases professional experience, skills, projects, and contact details in a modern, responsive format.

## 🛠️ Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript**
- **Three.js** for the animated 3D skills globe

## 🚀 Features
- Responsive layout optimized for desktop and mobile
- Interactive 3D globe with locally hosted Three.js
- Smooth scroll navigation and section reveal animations
- Contact area and resume download CTA
- Clean visual design with accessible mobile menu

## 📁 Folder Structure
```
/ (root)
  index.html
  index.css
  index.js
  README.md
  .gitignore
  /img
  /assets/lib/three
```

## 📦 Local Assets
The project includes local Three.js files under `assets/lib/three/` so the globe works offline without relying on CDN delivery.

## 📤 Upload to GitHub
1. Create a new repository on GitHub.
2. Initialize the local project folder:
   ```bash
git init
git add .
git commit -m "Initial portfolio upload"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```
3. Optionally enable GitHub Pages from the repository settings to publish the site.

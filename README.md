# 🚀 Kamalraj Portfolio

> A modern, responsive developer portfolio showcasing my professional journey, technical expertise, featured projects, and experience as a Software Engineer.

🌐 **Live Website:** https://kamalraj-s.github.io/Kamalraj-Portfolio/

---

# 📖 Overview

This portfolio serves as my personal website, highlighting my experience, technical skills, projects, certifications, and career achievements in a clean, interactive, and responsive interface.

The website is designed to provide recruiters, hiring managers, and fellow developers with an engaging overview of my work while demonstrating modern frontend development practices.

---

# ✨ Features

* 🎨 Modern and responsive UI
* 🌍 Interactive 3D Skills Globe built with Three.js
* ⚡ Smooth scrolling and scroll reveal animations
* 📱 Mobile-friendly design
* 💼 Professional experience timeline
* 🚀 Featured projects section
* 📄 Resume download
* 📬 Contact section
* 🌙 Clean and minimal user interface
* 📈 Optimized for GitHub Pages deployment

# 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/Kamalraj-S/Kamalraj-Portfolio.git
```

Navigate to the project directory:

```bash
cd Kamalraj-Portfolio
```

Open `index.html` in your preferred browser.

No build tools or package installation are required.
